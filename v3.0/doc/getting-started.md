# Get Started

> An awesome project.

## Installing

1. Depend on it

Add this to your package's pubspec.yaml file:

```yaml
dependencies:
  bonfire: ^3.x.x
```

2. Install it

You can install packages from the command line:

```
$ flutter pub get
```

3. Import it

Now in your Dart code, you can use:

```dart
import 'package:bonfire/bonfire.dart';
```
